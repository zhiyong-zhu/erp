# 采购真实环境验收清单

目标：用真实环境跑通 `采购单 -> 到货验收 -> 正式入库 -> 库存余额 -> 库存流水 -> 采购退货 -> 应付统计 -> 异常记录`，并记录单据、库存账和金额账。

## 1. 验收前准备

- 后端已启动并完成 Liquibase 迁移。
- Web 端已登录具备采购、库存、原料权限的账号。
- 已存在供应商、原料、供应商报价。
- 已存在默认仓库 / 默认库位，库存余额服务可写入仓库、库位、批次字段。

## 2. 页面操作路径

| 步骤 | 页面 | 操作 | 期望结果 | 数据账记录 |
| --- | --- | --- | --- | --- |
| 1 | 原料管理 / 原料列表 | 创建或选择待采购原料 | 原料可用于采购明细 | 记录原料编码、原料 ID |
| 2 | 原料管理 / 供应商报价 | 维护供应商报价 | 采购单可带出价格 | 记录供应商、报价 |
| 3 | 采购管理 / 采购单 | 创建采购单并审批 | 采购单进入已审批 | 记录采购单号、采购数量、采购金额 |
| 4 | 采购管理 / 采购单 | 执行到货验收 | 合格数量入库，不合格数量生成异常 | 记录收货数量、合格数量、拒收数量 |
| 5 | 库存管理 / 正式入库单 | 查询采购入库单 | 生成正式入库单 | 记录入库单号 |
| 6 | 库存管理 / 库存流水 | 查询采购入库流水 | 出现 `PURCHASE_IN` 流水 | 核对库存余额增加数量 = 合格数量 |
| 7 | 采购管理 / 采购退货 | 对已入库数量发起退货 | 生成采购退货单并扣减库存 | 记录退货单号、退货数量、退货金额 |
| 8 | 库存管理 / 库存流水 | 查询采购退货流水 | 出现 `PURCHASE_RETURN` 流水 | 核对库存余额减少数量 = 退货数量 |
| 9 | 采购管理 / 应付统计 | 查询供应商应付 | 应付按合格入库金额扣减退货金额 | 记录应付金额、退货金额、净应付 |
| 10 | 采购管理 / 采购异常 | 查询验收异常 | 不合格数量生成异常记录 | 记录异常单号、异常类型、状态 |

## 3. 数量与金额一致性口径

- 合格入库：库存余额增加数量必须等于采购验收合格数量。
- 拒收异常：拒收数量不入库，应生成采购异常记录。
- 退货扣减：采购退货必须扣减库存余额，并写 `PURCHASE_RETURN` 负向流水。
- 应付统计：应付金额 = 已合格入库数量 * 采购单价 - 已退货数量 * 采购单价。
- 单据追溯：库存流水必须能追溯到正式入库单或采购退货单。
